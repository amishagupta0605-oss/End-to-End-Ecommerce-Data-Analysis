# 🛒 End-to-End E-Commerce Data Analytics Project

## 📌 Overview

This project is a complete **end-to-end data analytics pipeline** built on an e-commerce dataset. It simulates real-world business scenarios where raw transactional data is transformed into meaningful insights that drive decision-making.

The project covers:

* Data cleaning & preprocessing
* SQL-based analysis (Basic → Advanced)
* Business insights generation
* Data visualization & storytelling

---

## 🎯 Objectives

* Analyze customer behavior and purchasing patterns
* Identify revenue drivers and high-performing categories
* Measure business growth and retention metrics
* Solve complex analytical problems using SQL and data analysis techniques

---

## 🛠️ Tech Stack

* **SQL** (PostgreSQL / MySQL)
* **Python** (Pandas, NumPy, Matplotlib / Seaborn) *(if used)*
* **Power BI / Tableau** *(for dashboards, if included)*
* **Excel** *(optional)*

## 🔍 Problem Statements & Analysis

### 🟢 Basic Analysis

* List all unique cities where customers are located
* Count the number of orders placed in 2017
* Calculate total sales per category
* Determine percentage of orders paid in installments
* Count customers from each state

---

### 🟡 Intermediate Analysis

* Monthly order trends for 2018
* Average number of products per order by city
* Revenue contribution by product category
* Correlation between product price and purchase frequency
* Seller-wise revenue ranking

---

### 🔴 Advanced Analysis

* Moving average of customer order values
* Cumulative monthly sales per year
* Year-over-year (YoY) growth rate
* Customer retention rate (repeat purchase within 6 months)
* Top 3 customers by spending each year

All queries are derived from real-world analytical scenarios 

---
